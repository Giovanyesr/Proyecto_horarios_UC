# Sistema de Generación Óptima de Horarios Académicos

## Descripción
Este proyecto consiste en el desarrollo de una aplicación web inteligente orientada a la generación automática de horarios académicos en entornos universitarios con currículo flexible. El sistema analizará múltiples variables y restricciones académicas, como la disponibilidad de docentes, estudiantes y aulas, con el fin de generar horarios válidos sin conflictos.

## Objetivo del proyecto
Diseñar e implementar una aplicación web que permita automatizar el proceso de generación de horarios académicos, optimizando la planificación académica y mejorando la gestión de recursos institucionales.

## Tecnologías utilizadas
- Frontend: React
- Backend: Node.js / Express
- Base de datos: PostgreSQL
- Control de versiones: Git y GitHub

## Estructura del proyecto
```

/backend     → API y lógica del sistema  
/database    → Scripts y modelos de base de datos

```
