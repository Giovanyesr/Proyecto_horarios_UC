# **PROJECT CHARTER**

## **1. Información general del proyecto**

**Nombre del proyecto:** Sistema de Generación de Horarios Académicos
Inteligente\
**Tipo de proyecto:** Desarrollo de aplicación web\
**Metodología:** Scrum\
**Fecha de inicio:** 23/03/2026\
**Fecha estimada de finalización:** 01/06/2026

## **2. Propósito del proyecto**

El propósito del proyecto es diseñar e implementar una aplicación web
inteligente capaz de generar horarios académicos óptimos de manera
automatizada, considerando múltiples restricciones académicas,
operativas y contextuales.

Este proyecto busca resolver las ineficiencias actuales en la
planificación académica, reduciendo errores, conflictos de horarios y
optimizando el uso de recursos institucionales.

## **3. Objetivos del proyecto**

### **3.1 Objetivo general**

Desarrollar una aplicación web inteligente que permita generar horarios
académicos óptimos de forma automatizada, mejorando la eficiencia en la
gestión académica.

### **3.2 Objetivos específicos**

- Analizar el problema de generación de horarios académicos
  identificando variables, restricciones y ambigüedades.

- Diseñar la arquitectura del sistema web.

- Implementar un algoritmo de generación de horarios basado en técnicas
  de optimización.

- Desarrollar una interfaz web intuitiva para la gestión de datos y
  visualización de horarios.

- Validar el sistema mediante pruebas funcionales y de rendimiento

## **4. Alcance del proyecto**

### **4.1 Alcance incluido**

El proyecto incluirá:

- Registro de docentes, cursos y aulas

- Definición de disponibilidades y restricciones

- Generación automática de horarios académicos

- Detección de conflictos

- Visualización de horarios por diferentes criterios

- Exportación de horarios

### **4.2 Alcance excluido**

El proyecto no incluirá:

- Integración con sistemas institucionales existentes

- Implementación de aplicaciones móviles

- Soporte para múltiples instituciones en una misma plataforma
  (multi-tenant)
